Projects

Project 01 · EDA

ADMET Properties: Exploratory Data Analysis

A thorough EDA of drug physicochemical properties. Applies Lipinski's Rule of Five to assess oral drug-likeness, explores distributions of MW, LogP, TPSA, and bioavailability, and uncovers feature correlations and toxicity patterns.

Pandas Seaborn Matplotlib Lipinski Ro5 Statistics

Techniques Used

Histograms & KDE
Correlation Heatmap
Boxplots by Class
Violin Plots

Key Findings

~70% of compounds pass Lipinski's Rule of Five
LogP negatively correlates with solubility
Toxic compounds show higher MW and LogP

Project 02 · Classification

Drug Toxicity Prediction with Machine Learning

Builds and benchmarks three classification models — Logistic Regression, Random Forest, and SVM — to predict drug toxicity from ADMET features. Evaluates via ROC-AUC, confusion matrices, and cross-validation.

Scikit-learn Random Forest SVM ROC-AUC Cross-Validation

Techniques Used

Logistic Regression
Random Forest
Feature Importance
ROC Curves

Key Findings

Random Forest achieved best AUC across all models
Bioavailability & LogP were top toxicity predictors
5-fold cross-validation confirmed model stability

Project 03 · Regression + Clustering

Bioavailability Prediction & ADMET Profiling

Predicts continuous oral bioavailability (%) using regression models and clusters compounds by their full ADMET profile using K-Means + PCA. Delivers a complete drug profiling pipeline applicable to early-stage drug screening.

Ridge Regression K-Means PCA R² RMSE

Techniques Used

Linear & Ridge Regression
K-Means Clustering
PCA Visualization
Elbow Method

Key Findings

Random Forest best predicted bioavailability (R²)
K=3 clusters revealed distinct ADMET drug profiles
TPSA & MW were strongest bioavailability predictors
